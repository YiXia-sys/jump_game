# 其他游戏模块 — 需求文档（PRD）

## 一、项目概述

在现有跳跳游戏平台中新增7款经典休闲小游戏，通过"其他游戏"入口进入。每款游戏独立为单个 HTML 文件，纯前端 Canvas 实现，无需后端支持。所有游戏共享跳跳游戏的视觉风格（深色主题、霓虹配色）。

## 二、技术方案

- 每个游戏一个独立 HTML 文件，放在 `/games/` 目录下
- 纯 Canvas + 原生 JS，无第三方依赖
- 通过 `server.js` 提供静态文件服务（已有 express.static）
- 移动端优先，支持触屏操作
- 每个游戏页面包含"返回"按钮，跳转回主页面
- 统一深色背景 + 霓虹风格 UI

## 三、游戏入口

"其他游戏"页面展示7个游戏卡片，每个卡片包含：
- 游戏图标（emoji）
- 游戏名称
- 一句话描述
- 点击跳转到对应游戏页面

### 游戏1：🐝 小蜜蜂（太空侵略者）

**文件**: `games/space-invaders.html`

**玩法**:
- 玩家控制底部飞船，左右移动，**自动射击**（玩家只需操控移动）
- 敌人排列成阵列（5行×8列），整体左右移动，到边缘后下移一行
- 敌人随机向下射击
- 消灭所有敌人过关，进入下一波（速度加快）
- 玩家3条命，被击中减一条

**操作**:
- 移动端：手指左右拖动/滑动控制飞船位置（自动射击，无需射击按钮）
- PC端：方向键左右 或 鼠标左右移动（自动射击）

**核心功能**:
- [ ] 玩家飞船移动（自动射击，射速随波次可提升）
- [ ] 敌人阵列移动逻辑（左右+下移）
- [ ] 碰撞检测（子弹vs敌人、敌人子弹vs玩家）
- [ ] 分数系统（不同行敌人不同分值）
- [ ] 多波次递进难度
- [ ] 生命值显示
- [ ] 游戏结束/重新开始

## 六、爽感设计（全局）

所有游戏必须在关键时刻给予玩家强烈的正反馈，让人"停不下来"。以下为通用爽感机制和各游戏专属设计：

### 通用爽感机制（每个游戏都要有）

| 机制 | 说明 |
|------|------|
| 屏幕震动 | 关键击杀/消除/得分时画面短暂抖动（2-4帧，幅度3-5px） |
| 粒子爆炸 | 消灭敌人/消除方块/吃到食物时喷射彩色粒子（15-30个，带重力和衰减） |
| 连击计数 | 短时间内连续得分显示 "x2 COMBO!" "x3 COMBO!" 等，字体放大+渐隐 |
| 分数飘字 | 得分时在事件位置弹出 "+100" 等飘字，向上漂浮并淡出 |
| 闪光高亮 | 关键事件瞬间全屏白色闪光（opacity 0.3，快速衰减） |
| 慢动作 | 关键时刻（最后一个敌人、险胜）短暂降速 50ms，营造"子弹时间"感 |
| 里程碑庆祝 | 达到特定分数/关卡时全屏撒花/彩带动画 |

### 各游戏专属爽感设计

**🐝 小蜜蜂**:
- 连续击杀时射速临时加快，子弹变大变亮
- 一排敌人全灭时整行爆炸+大粒子特效
- 波次清空时 "WAVE CLEAR!" 大字+全屏闪光
- 最后一个敌人击杀慢动作+特写爆炸

**🐍 贪吃蛇**:
- 吃食物时蛇身闪光波纹从头到尾传递
- 连续快速吃到食物时屏幕边缘发光（速度线效果）
- 每10个食物触发"狂热模式"：3秒内食物密集出现，背景变色加速
- 蛇身长度里程碑（20/50/100）触发庆祝特效

**🏓 弹球打砖块**:
- 球击中砖块时砖块碎裂粒子飞溅
- 连续消砖不落地时 combo 计数，球拖尾变长变亮
- 一次弹射消多个砖块时触发"多重打击"特效
- 清关时所有剩余砖块依次爆炸（连锁爆炸效果）

**💣 扫雷**:
- 翻开安全格时格子展开有弹性动画
- 连锁翻开大片空白区域时有"波纹扩散"动画效果
- 标旗时旗帜插入有弹跳动画
- 胜利时所有雷依次变成烟花爆炸

**🎮 坦克大战**:
- 击毁敌方坦克时火焰爆炸+残骸碎片飞溅
- 连续击杀显示 "DOUBLE KILL!" "TRIPLE KILL!" 等
- 子弹穿过草丛时有草叶飘散效果
- 砖墙被打碎时碎块飞溅物理效果
- 过关时剩余生命值转化为奖励分，逐个跳动累加

**🔲 俄罗斯方块**:
- 消行时该行闪白后粒子化消散
- 同时消2行以上时屏幕震动加强，"DOUBLE!" "TRIPLE!" "TETRIS!" 大字
- 消4行（Tetris）时全屏闪光+彩虹粒子雨
- 方块硬降时落点产生冲击波纹
- 连续消行 combo 时背景颜色渐变加速

**🏃 跑酷闯关**:
- 险些撞到障碍物时触发"Near Miss"特效+加分
- 跑步时脚下有尘土粒子
- 跳跃时有拉伸压缩的弹性动画（起跳压扁，空中拉长）
- 每1000米里程碑时背景主题切换+庆祝文字
- 速度越快屏幕两侧速度线越密集

## 七、关卡/难度设计

### 🐝 小蜜蜂 — 波次制（无限）

| 波次 | 敌人阵列 | 敌人速度 | 敌人射击频率 | 自动射速 | 特殊机制 |
|------|---------|---------|------------|---------|---------|
| 1 | 3行×6列 | 慢 | 低（2秒/发） | 0.5秒/发 | 无 |
| 2 | 4行×7列 | 中慢 | 中低（1.5秒） | 0.45秒 | 无 |
| 3 | 5行×8列 | 中 | 中（1.2秒） | 0.4秒 | 出现1个精英敌人（2血，红色） |
| 4 | 5行×8列 | 中快 | 中高（1秒） | 0.35秒 | 精英敌人增多，偶尔出现UFO飞过顶部（击中高分） |
| 5 | 5行×9列 | 快 | 高（0.8秒） | 0.3秒 | 敌人下移幅度加大 |
| 6+ | 5行×10列 | 递增 | 递增 | 0.25秒封顶 | 每3波出现BOSS（大型敌人，10血，多方向射击） |

- 每波清空后有2秒休息 + "WAVE X CLEAR!" 动画
- 每5波奖励一条额外生命（最多5条）
- BOSS波：单独一个大型敌人，左右移动+多弹幕，击杀后大量粒子+奖励分

### 🐍 贪吃蛇 — 无限模式（速度递增）

| 蛇长度 | 移动间隔 | 特殊事件 |
|--------|---------|---------|
| 1-10 | 200ms | 普通食物 |
| 11-20 | 170ms | 偶尔出现金色食物（+3分，限时5秒消失） |
| 21-35 | 140ms | 出现障碍方块（随机放置1-2个，不可穿越） |
| 36-50 | 120ms | 金色食物更频繁，障碍增至3个 |
| 51-70 | 100ms | 出现毒食物（紫色，吃到减3节但不死） |
| 71+ | 85ms封顶 | 全要素混合，障碍最多5个 |

- 每吃10个食物触发一次"狂热模式"（3秒内连续出现3个食物，背景加速闪烁）
- 无关卡概念，纯粹追求最高分

| 关卡 | 砖块布局 | 砖块耐久 | 球速 | 特殊砖块 |

| 2 | 金字塔形 | 1-2血混合 | 慢 | 无 |
| 3 | 菱形 | 1-2血 | 中 | 出现银色砖块（2血） |
| 4 | 两个三角形 | 1-3血 | 中 | 出现金色砖块（3血） |
| 5 | 心形 | 混合 | 中 | 出现道具砖块（掉落加长挡板/多球/穿透球） |
| 6 | 笑脸 | 混合 | 中快 | 钢砖出现（不可破坏，球反弹） |
| 7 | 交错条纹 | 2-3血为主 | 中快 | 道具砖块增多 |
| 8 | 螺旋形 | 混合 | 快 | 爆炸砖块（消除时炸掉周围砖块） |
| 9+ | 随机生成 | 递增 | 递增 | 全要素混合 |

- 道具类型：挡板加长（10秒）、多球分裂（1变3）、穿透球（穿过砖块不反弹，5秒）、减速球（5秒）

- 第9关起为无限随机关卡
